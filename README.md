## General info
A Weather App which shows the weather conditions of a specific typed out city
	
## Technologies
Project is created with:
* Bootstrap v5.0
* Ajax 
* NodeJs v14.15.0
* npm v6.14.8
* Express ^4.17.1v
* Pug ^3.0.2v
* Nodemom ^2.0.7v

	
## Setup
To run this project, install it using npm:

```
$ npm install
$ npm run dev
```
or

```
$ npm install
$ npm run start
```
